# Estrutura-de-Dados---Projeto-de-Hashing

Projeto da disciplina Estrutura de Dados Lineares <br>
Projeto de autenticação utilizando tabela hash em C. <br>
Integrantes: Danilo Teixeira, Eduardo Rocha e Nathan Mansur


## Funcionalidades

- Cadastro de usuários
- Login
- Remoção de usuários
- Impressão da tabela hash
- Tratamento de colisões com lista encadeada

---

## Tecnologias Utilizadas

- Linguagem C
- GCC

---

## Requisitos

Para executar o projeto é necessário:

- Compilador GCC
- Terminal Linux, WSL ou GitHub Codespaces

---

## Estrutura

```text
.
├── main.c
├── hash.c
├── hash.h
```

---

## Compilação e Execução

Compile o programa com:

```bash
gcc main.c hash.c -o programa
```

Execute com:

```bash
./programa
```
